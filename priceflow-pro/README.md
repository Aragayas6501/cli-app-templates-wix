# PriceFlow Pro

PriceFlow Pro is a Wix CLI app template for quantity-based pricing in Wix Stores. It includes a dashboard rule manager, Wix discount-rule sync, a checkout Discount Triggers service plugin, app-owned data collections, and a storefront site plugin that displays tiered pricing on product pages.

## What it demonstrates

- Wix dashboard page and dashboard modal for rule management.
- Wix Data collections for app-owned pricing rules, analytics events, settings, and audit logs.
- Wix discount-rule sync plus a Discount Triggers service plugin for checkout-time eligibility.
- Storefront site plugin for live price and tier display on old and new Wix Stores product pages.
- Backend web methods for typed dashboard-to-backend calls.
- eCommerce event handlers for analytics ingestion.

## App namespace

Data collections use the placeholder namespace `<app-namespace>` in runtime collection IDs. Replace it with your app namespace from Wix Dev Center before publishing this app as a real Wix app.
